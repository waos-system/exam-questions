## SAMPLE QUESTIONS

### Question 001: Generic Type Declaration
**Topic**: Basic generics
**Question**: 次のコードで、Listの型パラメータとして正しい宣言はどれか。
**Answer**: List<Object> list = new ArrayList<Object>();
**Explanation**: Type parameters must match on both sides; Object is the parent of all classes, making this the only valid option.

### Question 020: Reduce Operation
**Topic**: Stream reduction
**Question**: reduce操作の正しい使用法はどれか。
**Answer**: All options are correct (various forms of reduce are valid)
**Explanation**: Reduce supports multiple signatures with different accumulator patterns.

### Question 042: Collectors.groupingBy()
**Topic**: Stream collectors
**Question**: Collectors.groupingBy()の用途として正しいものはどれか。
**Answer**: Groups stream elements by key into a Map
**Explanation**: groupingBy creates a Map with groups of elements based on the key function.
