# 📚 QuizForge — 試験問題集サイト

試験の問題を繰り返し解いて合格を目指す、静的ファイルのみで動く問題集サイトです。

## ✨ 機能

- 🎯 **問題数を選べる** — 5・10・15・20問またはカスタム入力
- 📱 **モバイル対応** — スマホからも快適に操作
- 📖 **解説付き** — 正誤判定後に詳しい解説を表示
- 🏆 **結果表示** — 正解率・正解数を視覚的に表示
- ➕ **簡単に問題追加** — JSONファイルを追加するだけ

## 🚀 ローカルで起動

```bash
# Node.js を使う場合（推奨）
npx serve .

# Python を使う場合
python3 -m http.server 8080
```

ブラウザで `http://localhost:3000` (or 8080) を開く。

> ⚠️ `index.html` をダブルクリックで直接開くとJSONが読み込めません。必ずサーバ経由で起動してください。

## ➕ 問題を追加する方法

### 1. JSONファイルを作成
```json
{
  "genre": "ジャンル名",
  "exam": "試験名",
  "questions": [
    {
      "id": "xx_001",
      "question": "問題文",
      "choices": ["選択肢A", "選択肢B", "選択肢C", "選択肢D"],
      "answer": 0,
      "explanation": "解説文"
    }
  ]
}
```

### 2. `data/genres.json` に登録
```json
{
  "id": "new_section",
  "name": "新しいセクション",
  "icon": "📘",
  "description": "説明",
  "file": "data/folder/questions_xxx.json",
  "count": 10
}
```

詳細は `QUESTION_TEMPLATE.txt` と `SETUP.md` を参照。

## 📁 ファイル構成

```
.
├── index.html              ← サイト本体
├── data/
│   ├── genres.json         ← ジャンル定義（階層管理）
│   ├── fe/                 ← 基本情報技術者試験
│   │   └── questions_fe_*.json
│   └── english/            ← 英語
│       └── questions_en_*.json
├── SETUP.md                ← 環境構築ガイド
├── QUESTION_TEMPLATE.txt   ← AI問題作成テンプレート
└── README.md               ← このファイル
```
